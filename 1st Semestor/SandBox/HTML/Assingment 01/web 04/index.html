Left Side

Div tag is used for grouping many HTML tags like tables, lists, paragraphs in a web page. Mostly, div tags are used in web pages to partition or make a section for content or HTML tags. To make a particular design for some special content, the div tag is useful because it does not affect the div layout and any CSS design.
